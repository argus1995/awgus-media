Ini adalah contoh konten menggunakan **Markdown**. Karena kita menggunakan struktur folder kategori, file ini berada di dalam `src/content/posts/tech/`.

### Mengapa struktur ini bagus?

1. **SEO Friendly**: URL akan menjadi `/tech/my-first-post`.
2. **Terorganisir**: Gambar tersimpan rapi di folder assets.
3. **Mudah Dikelola**: Menambahkan post baru semudah membuat file `.md` baru.

Anda bisa menulis konten apa saja di sini, termasuk kode:

```javascript
console.log("Astro sangat cepat!");
```
